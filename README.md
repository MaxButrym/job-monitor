# Job Monitor

Telegram-бот для мониторинга вакансий с API-интерфейсом.

## Структура проекта

- `bot/` — логика Telegram-бота.
- `parser/` — парсеры для сбора вакансий.
- `database/` — работа с базой данных.
- `api/` — FastAPI endpoints.
- `services/` — бизнес-логика.
- `config.py` — настройки проекта.
- `main.py` — точка запуска приложения.

## Запуск

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Настройте переменные окружения (см. `config.py`).
4. Запустите приложение: `python main.py`.
